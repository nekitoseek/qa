- Создать объявления - https://qa-internship.avito.com/api/1/item
- Получить объявления по его идентификатору - https://qa-internship.avito.com/api/1/item/тутidобъявления
- Получить все объявления по идентификатору продавца - https://qa-internship.avito.com/api/1/тутidпользователя/item
- Получить статистику по айтем id - https://qa-internship.avito.com/api/1/statistic/тутidобъявления

### 1. Тесты на создание объявления (`POST /api/1/item`)

Позитивные тесты

| ID      | Название                                                       | Шаги                                                                 | Ожидаемый результат                                   |
| ------- | -------------------------------------------------------------- | -------------------------------------------------------------------- | ----------------------------------------------------- |
| POST-01 | Создание объявления с валидными данными                        | Отправить POST с корректным `sellerId`, `name`, `price`              | Код 200, в ответе вернётся `id` созданного объявления |
| POST-02 | Создание объявления с одинаковым `name`, но разными продавцами | Создать два объявления с одинаковым названием, но разными `sellerId` | Код 200, создаются разные объявления                  |
| POST-03 | Создание объявления с максимальной длиной name                 | name = 255 символов                                                  | Код 200, объявление создаётся                         |
| POST-04 | Создание двух объявлений с одинаковыми данными                 | Создать два одинаковых объявления                                    | Код 200, оба объявление имеют разные id               |

Негативные тесты

| ID      | Название                                            | Шаги                   | Ожидаемый результат |
| ------- | --------------------------------------------------- | ---------------------- | ------------------- |
| POST-05 | Пустой JSON                                         | Отправить {}           | 400 Bad Request     |
| POST-06 | Отправка без sellerId                               | Опустить поле sellerId | 400                 |
| POST-07 | Отправка без name                                   | Опустить name          | 400                 |
| POST-08 | Использовать sellerId вне диапазона (меньше 111111) | sellerId = 100         | 400                 |
| POST-09 | Использовать sellerId вне диапазона (больше 999999) | sellerId = 10000000    | 400                 |
| POST-10 | Некорректный тип данных для sellerId (строка)       | sellerId = "abc"       | 400                 |
| POST-11 | Некорректный тип name (число)                       | name = 123             | 400                 |
| POST-12 | Создание объявления с пустым name                   | name = ""              | 400                 |
| POST-13 | Очень длинный name (>255)                           | name = 1000 символов   | 400 или 413         |

| ID     | Шаги                                               | Ожидаемый результат                        |
| ------ | -------------------------------------------------- | ------------------------------------------ |
| GET-01 | Создать объявление, затем получить по id           | 200, корректный JSON, все данные совпадают |

Негативные тесты

| ID     | Шаги                                         | Ожидаемый результат |

| GET-04 | Использовать некорректный формат id (строка) | 400                 |
| GET-05 | id = пустая строка                           | 404                 |
